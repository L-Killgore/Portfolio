MY PROJECTS

Below are three web apps I created, the first of which is a React app made using the PERN Stack, and the other two are Django apps that were adapted from assignments from Harvard's CS50: Web Programming with Python and JavaScript course, for which I received certification viewable in my About Me section below. If you wish to demo one of the apps, an account can be made on Prediction Tracker, or a separate account can be made on either of the two Django projects. All of my code is viewable on GitHub by clicking the "View Code" buttons next to each project.

These sites are still in development and may change over time. They are for showcasing purposes only! Do not share any personal information on them!

Prediction Tracker

An app where users can share their predictions.

On Prediction Tracker, users can post predictions they would like to make, give reasons to support their predictions, and view the predictions of others. Each Prediction can be voted on as to its plausbility or correctness.

Specifications: This full-stack App was created using React, Node.js, and Express with a PostgreSQL database (PERN Stack). Sequelize is used as the ORM tool, in addition to Axios used for handling API calls. Finally, React's createContext() hook is used to create a Context Provider to handle state globally. Heroku is the service provider that serves the backend of the app, and an AWS S3 Bucket is used to serve the frontend. Bootstrap is used for much of the design.

Auction House

A simple e-commerce website.

Users can create listings, bid on items, add items to a watchlist, and post comments on listings. The poster of a listing can also close the bidding for the item, and the winner will be displayed. Lastly, each listing is put in a category that is browsable through the Categories page.

Specifications: This app was built using Django and Bootstrap, along with an AWS S3 Bucket for serving static files.

Coding Corner

A forum website.

At Coding Corner, users can create topics on forum boards, make posts on topics, and comment on posts. The categories and board titles can only be set by a user with administrative privileges, but any user can start a new topic. Post and View counts are tallied, and information about the most recent post in a topic is displayed. This project also has a user profile with editable content, along with a link that leads to a page with a list of all the topics and posts a user has made. Lastly, an administrator can upload banner images and forum titles and swap these on the fly.

Specifications: This app was built using Django and Bootstrap, along with an AWS S3 Bucket for serving static files.
